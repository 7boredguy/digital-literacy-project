## About This Project

This portfolio has been prepared for the CSE0001 Digital Literacy course at VIT Bhopal University. <br>
The main aim of this project is to help first-year students learn how to use digital tools in a proper and responsible way, develop a professional presence online, and understand how to stay safe on the internet. <br>
All the work has been arranged in different folders based on each task for better organization.

## Module Summaries

### Task 1: Digital Literacy Infographic
In this task, I designed a one-page infographic that explains the meaning of digital literacy, highlights some useful tools for students, and gives basic ideas on building a professional online presence. I used Canva for designing because it is simple to use and provides many free templates.

### Task 2: Student Digital Portfolio
For this task, I created profiles on LinkedIn, GitHub, and Kaggle. I had not used LinkedIn properly earlier, so this task helped me understand its purpose and allowed me to complete my profile with my academic details.

### Task 3: Coding and Collaboration Platforms
I signed up on HackerRank and completed a beginner-level coding challenge. Along with this, I also created a Digital Literacy Quiz with 5 questions using Google Forms, which can be used to test other students.

### Task 4: Email Etiquette
In this task, I drafted two formal emails. One email was written to a professor requesting an extension on a deadline, and the other was addressed to an internship coordinator. I also prepared a checklist covering do’s and don’ts of social media for college students.

### Task 5: Cybercrime Awareness
For this task, I studied phishing attacks and created a case study explaining how such scams happen and who are commonly targeted. I also made a prevention checklist with tips for college students in India, including safety points related to UPI.
